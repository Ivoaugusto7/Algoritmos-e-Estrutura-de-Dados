# Tabela da Verdade Recursiva

O objetivo desta atividade é gerar a tabela da verdade para um conjunto de variáveis booleanas usando recursividade.

## Descrição

O programa deve gerar todas as combinações possíveis de valores booleanos para um conjunto de variáveis, respeitando a ordem binária crescente.

## Requisitos do Programa

- Ler um número inteiro **N** (`1 ≤ N ≤ 10`) representando a quantidade de variáveis booleanas.
- Gerar todas as combinações possíveis de valores (`0` ou `1`) para essas variáveis.
- A geração das combinações deve ser feita **obrigatoriamente de forma recursiva**.
- A saída deve exibir cada linha da tabela com os valores separados por espaço.
- Utilizar `0` para representar **falso** e `1` para representar **verdadeiro**.
- As linhas devem ser exibidas em **ordem binária crescente** (da combinação com todos os valores iguais a `0` até todos iguais a `1`).

## Formato de entrada

Um único número inteiro **N**.

## Formato de saída

Cada linha contendo **N** valores (`0` ou `1`), separados por espaço, representando uma combinação da tabela da verdade.
